# Лабораторная работа: Стандартный ввод-вывод

## Автор
Сайбуу Вилена

## Цель
Освоение механизмов сериализации, работы с файловой системой, реализации паттерна «Memento» для отмены изменений, а также создание простейшего поискового индекса в приложении на C#.

## Описание
Разработан комплекс классов для работы с текстовыми файлами:
- Класс `TextFile` с поддержкой бинарной и XML сериализации.
- Класс `FileSearcher` для поиска файлов по ключевым словам.
- Консольный редактор с историей изменений (паттерн Memento).
- Приложение для индексации файлов в директории.
  
## Ожидаемый результат
Приложение демонстрирует:
- Создание, редактирование и сохранение текстовых файлов.
- Поиск файлов по ключевым словам в указанной директории.
- Возможность отмены (undo) последних изменений в редакторе.
- Вывод индексации: список файлов и найденные ключевые слова.
